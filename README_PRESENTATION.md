# ExpertERPHub - Présentation Professionnelle Entreprises 2026

## Vue d'ensemble

Présentation PowerPoint professionnelle et premium pour ExpertERPHub, conçue pour les réunions avec des entreprises partenaires potentielles (ESN, cabinets ERP, sociétés de conseil).

- **Fichier principal:** `ExpertERPHub_Presentation_Entreprises_2026.pptx`
- **Durée:** 12-15 minutes de présentation + 5-10 minutes de questions
- **Nombre de slides:** 13
- **Format:** 16:9 PowerPoint (compatible universel)
- **Taille:** 46 KB

---

## Fichiers inclus

### 1. ExpertERPHub_Presentation_Entreprises_2026.pptx
La présentation PowerPoint finale, prête à l'emploi.

**Comment l'utiliser:**
1. Télécharger le fichier
2. Ouvrir dans PowerPoint, Google Slides, ou Keynote
3. Naviguer au mode présentation (F5 ou Cmd+Shift+P)
4. Présenter!

### 2. PRESENTATION_NOTES.md
Guide complet de présentation avec:
- Notes orales détaillées pour chaque slide
- Timing recommandé
- Tone et points clés à mémoriser
- Questions courantes et réponses
- Tips de présentation pro

**Utilisation:**
- Imprimer et annoter avec vos notes personnelles
- Lire avant la présentation pour mémoriser
- Relire 30 minutes avant de présenter

### 3. QA_CHECKLIST.md
Checklist complète de vérification qualité:
- Vérification du contenu (chaque slide)
- Vérification du design (couleurs, typographie, layout)
- Vérification technique (format, compatibilité)
- Vérification métier (alignement avec business model)
- Sign-off et prochaines étapes

**Utilisation:**
- Avant chaque présentation, passer la checklist
- Pour valider que tout est en place
- Pour identifier les ajustements avant la réunion

### 4. generate-presentation.py
Script Python qui génère la présentation à partir de zéro.

**Pourquoi:**
- Vous modifiez les couleurs? Exécuter le script avec les nouvelles couleurs
- Vous ajoutez du contenu? Modifier le script et régénérer
- Vous versionnez le code? Le script est dans git

**Comment l'utiliser:**
```bash
python3 generate-presentation.py
```

Cela génère `ExpertERPHub_Presentation_Entreprises_2026.pptx`

## Design & Palette

- **Primary Navy:** #1B2A4A (texte, titres)
- **Secondary Gold:** #D4AF37 (accents, CTA, bars)
- **Accent White:** #FFFFFF (fond clair, texte blanc)
- **Dark Background:** #0F1419 (slides titre/conclusion)
- **Light Background:** #F5F7FA (slides contenu)
- **Text Muted:** #64748B (légende, info secondaire)
- **Accent Blue:** #3B82F6 (boîtes info)

### Éléments visuels
- Gold accent bar (0.12") en haut de chaque slide contenu
- Boîtes arrondies avec shadows pour profondeur
- Cercles numérotés pour processus
- Grilles pour données (systèmes ERP)
- Colonnes pour comparaisons

### Typography
- **Titres:** Arial 40pt Bold (navy)
- **Corps:** Arial 16pt (navy)
- **Sous-titres:** Arial 18pt Bold (navy)
- **Muted:** Arial 12pt Italic (slate)

### Layout
- Format 16:9 (10" x 5.625")
- Margins 0.5" minimum
- Spacing 0.3-0.5" entre sections
- Aucun mur de texte - visuels d'abord

## Utilisation et Personnalisation

### Utilisation standard
1. Ouvrir le fichier PPTX
2. Naviguer slides
3. Présenter!

**Pas d'installation requise.** Le fichier est auto-contenu. Aucune police manquante (Arial + système fonts).

### Personnaliser le contenu
**Option 1 - Directement dans PowerPoint:**
1. Ouvrir le PPTX
2. Cliquer sur le texte
3. Éditer
4. Sauvegarder

**Option 2 - Régénérer depuis Python:**
1. Éditer `generate-presentation.py`
2. Modifier le contenu/couleurs
3. Exécuter: `python3 generate-presentation.py`
4. Ouvrir le nouveau fichier généré

### Ajouter du contenu
**Pour ajouter une slide:**
1. Option A: Duplifier une slide dans PowerPoint (Ctrl+D)
2. Option B: Ajouter une fonction dans le script Python et régénérer

**Pour modifier les couleurs:**
1. Ouvrir `generate-presentation.py`
2. Modifier le dictionnaire `colors = {...}`
3. Exécuter le script
4. Voilà!

## Questions courantes

### "Ça s'ouvre comment?"
Ouvrir le fichier `.pptx` avec:
- Microsoft PowerPoint (recommandé)
- Google Slides (gratuit, web-based)
- Apple Keynote (Mac)
- LibreOffice Impress (gratuit, open-source)

### "Ça manque des fonts?"
Non. La présentation utilise uniquement Arial, une font système. Aucun fallback requis.

### "Je veux ajouter mon logo?"
Ouvrir le PPTX dans PowerPoint → Insert → Image → Uploader votre logo

### "Je veux changer les couleurs?"
2 options:
1. Éditer directement dans PowerPoint (Format → Colors)
2. Modifier `generate-presentation.py` et régénérer

### "Comment imprimer?"
File → Print → Sélectionner "Full Page Slides" → Print

### "Ça fonctionne en mode presenter?"
Oui. F5 pour présenter. Les notes apparaissent sur votre écran, les slides sur le projecteur.
